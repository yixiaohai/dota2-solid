# solid-panorama
此模版基于[solid-panorama](https://github.com/RobinCodeX/solid-panorama).

# 开始
## 克隆项目
会自动将项目链接至dota2项目文件夹，启动前确保dota2中没有同名的其他项目。
```
git clone https://github.com/yixiaohai/solid-template <你的项目名>
```
## 启动:
```
pnpm start
```
