# 说明

输入框的应用场景比较少，因为框架中默认面板不会聚焦，所以单独封装一下
可能应用场景：激活码、快捷键

# 参数

| 参数              | 说明                             | 类型                | 默认值 |
| ----------------- | -------------------------------- | ------------------- | ------ |
| type              | 预设样式类型：1-，2-             | 1 \| 2              | 1      |
| flow              | 均分占满容器宽度                 | boolean             | false  |
| style             | 自定义样式对象（会覆盖预设样式） | Partial<PanelStyle> | -      |
| ontextentrychange | text 改变时回调                  | Function            | -      |
