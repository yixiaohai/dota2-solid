# 使用

```ts
<CSlider />
```

# 说明

可以方便的设置横向样式

# 参数

| 参数           | 说明             | 类型                       | 默认值     |
| -------------- | ---------------- | -------------------------- | ---------- |
| min            | 最小             | number                     | 0          |
| max            | 最大             | number                     | 1          |
| value          | 初始值           | number \| Accessor<number> | -          |
| direction      | 横向还是竖向     | 'vertical' \| 'horizontal' | 'vertical' |
| onvaluechanged | value 改变时回调 | Function                   | -          |
